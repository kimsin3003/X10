## Ver. 0.09

### 1. Map Editor 구현
- target0.json내용 로드함
- 단축키 입력 후 왼쪽 클릭으로 Sprite 생성
- enemy : "e" , star : "s" , mirror : "m" , vertexMirror : "v" , bubble : "b"
- 오른쪽 버튼으로 삭제
- 휠로 확대/축소
- 선택후 키보드 방향키로 위치 미세조정
- target0.json에 저장
- 만든 맵 상에서 임시 play 가능

### 2. 스테이지 선택 화면

- 스테이지 진행에 따른 콜렉션 획득 변경
- 획득한 콜렉션 클릭시 텍스트 상자 팝업
- 스테이지 추가 (5,6,7,8)
